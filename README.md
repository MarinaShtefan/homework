# Домашнее задание 6
Задание:
Вы получили задание от секретного агентства — написать программу, которая шифрует входящие сообщения по такому правилу.
Если количество символов в сообщении чётное, то сообщение надо разделить ровно посередине и первую половину поместить в конец, а вторую — в начало.
Если количество символов в сообщении нечётное, то каждый второй символ надо переместить в новую строку. После чего новую строку с перемещёнными символами приклеить в конец к первой.
